# OptimusTech - Página de Recrutamento

Projeto desenvolvido durante o desafio **7DaysOfCode da Alura**.

## 📌 Sobre o projeto

Uma empresa de tecnologia precisava de uma página para divulgar oportunidades de trabalho e atrair novos talentos.

O objetivo foi desenvolver uma interface moderna para apresentação de vagas, cultura da empresa e informações para candidatos.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Layout responsivo
